# QuestKit

Embeddable gamification SDK built end-to-end on Cloudflare.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)
![Status: Under construction](https://img.shields.io/badge/Status-Under%20construction-orange)

</div>

QuestKit is an open-source gamification toolkit you can drop into any web app —
React component library plus a vanilla `<script>` embed — backed by a
Cloudflare-only API (Workers, D1, KV, R2, Queues, Durable Objects, Workers AI).
It is aimed at product teams who want missions, rewards, balances, and live
progress without standing up a dedicated game backend. The repository is in
early scaffold — the surface area is being built phase by phase and is not yet
ready for production use.

See [`instruction/work/plan.md`](instruction/work/plan.md) for the build plan
and current progress.
